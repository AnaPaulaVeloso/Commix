Sobre o Commix

O Commix é uma ferramenta de teste de penetração de código aberto que automatiza o processo de detecção e exploração de vulnerabilidades de injeção de comandos em aplicações web.

Desenvolvido em Python, o Commix oferece uma interface de linha de comando poderosa e flexível para testadores de segurança e pesquisadores.

Conceitos Fundamentais

O que é Injeção de Comandos?

Uma vulnerabilidade que permite a execução de comandos arbitrários no sistema operacional do servidor através de uma aplicação web vulnerável.

Como Funciona?

O atacante insere comandos maliciosos em parâmetros de entrada que são processados pelo sistema operacional do servidor.

Impacto

Pode resultar em comprometimento completo do servidor, acesso não autorizado e roubo de dados sensíveis.

# Instalação
git clone https://github.com/commixproject/commix.git
cd commix

# Uso básico
python commix.py -u "http://exemplo.com/page.php?id=1"

# Opções avançadas
python commix.py -u "http://exemplo.com/page.php?id=1" --batch --random-agent
